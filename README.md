# MVVM Example

A simple Swift MVVM project that fetches a bunch of Chuck Norris jokes, runs them through a validator, and displays them in a UICollectionView.

- ✅ Responsive UI
- ✅ Dependency injection
- ✅ Unit tests
- ✅ Network requests
- ✅ Error handling
- ✅ MVVM
